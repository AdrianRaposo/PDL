Trabajo Javascript-PdL grupo 81
A continuación se explicará el contenido de la carpeta que contiene todo el trabajo:

-El ZIP "Ejecutable.zip" contiene un archivo .jar que ejecuta directamente la interfaz grafica del trabajo. Además contiene todas las carpetas y archivos para su correcto funcionamiento. Para usar la interfaz es necesario hacer lo siguiente:
	-Escribir el código que se desea analizar, seleccionar el archivo.txt con el codigo que se quiere analizar o arrastrar el archivo .txt a la zona de arrastrar archivo de la interfaz.
	-A continuación guardar el archivo con el boton de guardar.
	-Una vez guardado seleccionar el boton de la lupa para analizar el código. Este guardará los archivos generados en la carpeta pruebasAnalizador.
	-Si se desean ver los archivos en la propia interfaz pulsar el botón inferior que dice Ver analisis si pasas el cursor por encima.

-El ZIP "Trabajo_Javascript_Pdl_Grupo_81.zip" incluye todos los scripts del codigo y los correspondientes archivos necesarios para la ejecución del mismo. En este caso se ha incluido además de la interfaz un script llamado "prueba.java" que se puede ejecutar como el main para realizar pruebas sin usar la interfaz.

-El Zip "Prueba_Trabajo.zip" incluye las pruebas realizadas en el anexo de la memoria con todos sus resultados en el formatos especificado.

-Los archivos "GramaticaSintactico.txt", "GramaticaSintacticoSemantica.txt" y "GramSinVersionMemoria.txt" contienen las distintas grámaticas usadas en los distintos ambitos del trabajo.

-Por último se incluye la tabla "TablaLL1.xlsx" por si no se apreciará bien en la memoria.
